# onealhtml

Cyberpunk portfolio with Matrix rain, glitch effects, and neon aesthetics.

## Stack

- Next.js 16 + React 19 + TypeScript
- Tailwind CSS 4

## Run

```bash
npm install
npm run dev
```

## Deploy

Push to GitHub → Deploy on [Vercel](https://vercel.com)
